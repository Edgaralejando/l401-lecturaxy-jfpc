Visualización de Datos X-Y con Flask

Edgar Díaz

Esta aplicación recibe datos X e Y desde una aplicación móvil y los muestra en una interfaz web desarrollada con Flask. La visualización utiliza cuatro cuadrantes para representar la posición actual según los valores recibidos. El cuadrante resaltado en color verde indica la ubicación correspondiente de los datos X e Y.
